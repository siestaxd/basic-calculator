# basic-calculator
Simple calculator application I made with python

# how to use
1 - install python<br>
2 - packages to install<br>
    ```python -m pip install customtkinter```<br>
    ```python -m pip install darkdetect```<br>
    ```python -m pip install Pillow```
    <br>
3 - start using or developing
